About the Site

On this site I've showcased some of the projects I've worked on outside of the office.

Choice of technologies

This site is set up using my favourite client side JavaScript framework, AngularJS. the site is a single page app which means although the initial load might take a bit longer, once it's there the site feels much more responsive.

I've used bootstrap 3 so I could quickly set up a good looking and responsive site.

Finally, I've used Highslide to provide image thumbnails and popup images.
