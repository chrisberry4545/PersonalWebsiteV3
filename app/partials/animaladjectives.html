Animal Adjectives

Animal Adjectives on the Play Store

Animal Adjectives started life as a game I played with my housemates while living in Canada. It's pretty silly but can be a tons of fun.

What does it do?

The applications consists of two lists. A list of animal names and a list of adjectives. It chooses a random word from each and combines the two together. If you've got an internet connection, it then goes to google images and gets the first image of the animal.

The core application is enhanced by features such as notifications (users are sent an regular Animal Adjectives), a favourites list and share to Facebook button.

Choice of technologies

I didn't need to look any further than the standard Android libraries for the majority of the application.

As the adjectives and animals are just stored in a text file they are easy to change and it was simple to implement. Using text files and getting the images from google rather than storing them locally has kept the size of the application down.

I used the Facebook API to allow users to share their favourite Animal Adjectives with their friends.

Challenges faced

I'd dabled with Android Apps before so much of it was quite familiar. The biggest problem I've had so far was getting the Share to Facebook button working. I had no experience of working with this and found it much more complicated than I expected. Getting things such as the test users set up and posting correctly so I could obtain permissions from Facebook were all long processes.

I also had some issues with notifications. Setting up a service is quite complicated and it took me some time to get the hang of it. I spent much longer implementing the additional features than the core app itself, which I guess is to be expected given how simple the core is.

The other problem I often find myself facing with Android is making the app look good on a variety of screen sizes. I have both a large tablet and smaller phone which made the task easier but it always seems quite fiddly. I find the process much easier with web technologies and I intend to see if I can recreate the app using PhoneGap, which may also mean I can release it for other operating systems.
