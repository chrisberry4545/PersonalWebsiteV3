As part of my MSc. in Computer Science I wrote a paper titled 'Using reinforcement learning to find a winning strategy for Monopoly. I received a distinction for this paper.

What does it do?

My project uses Reinforced Learning (an area of Machine Learning) and a Neural Network to allow an Artificial Intelligence to teach itself how to be a great Monopoly player.

The AI played many thousands of games against itself during which is tried to work out which moves are most likely to lead to success.

Choice of technology

The project was programmed using Java. This was the language I was most comfortable with at the time and also the language which the neural network my tutor gave me was written in. I used the GUI widget toolkit, Swing, for the display but the rest of the project was implemented using the standard Java libraries.

I used Java Web start to allow people to easily download the program and send me feedback on their experience with the AI.

Challenges faced

The biggest challenge faced was doing everything I wanted to in the time limit. As the AI had to play so many games against itself, getting a single version ready for testing would often take several days. I ran versions on my PC and on some Amazon Servers. I worked hard over the first few weeks to get the basic system constructed and well tested. I had to experiment with different inputs but due to the time it took I always needed to make sure I had the next experiment well planned based on the outcomes of the experiments I was currently running.

Much more detail about the experiments I tried and challenges faced can be found in the Dissertation itself.
